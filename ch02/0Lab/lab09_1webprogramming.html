강좌 소개

웹 프로그래밍 강좌에 오신 것을 환영합니다. 이 강좌는 웹 개발의 기초부터 고급 기술까지 다루는 종합 과정입니다.

HTML5 기초

HTML5는 웹 페이지의 구조를 정의하는 마크업 언어입니다. 이 과정에서는 다음 내용을 학습합니다:

  • HTML 문서 구조
  • 텍스트 서식 지정
  • 목록과 표 만들기
  • 시맨틱 태그 활용

CSS3 스타일링

CSS3는 웹 페이지의 디자인을 담당하는 스타일 시트 언어입니다. 다음 내용을 배웁니다:

  1. 선택자와 속성
  2. 박스 모델
  3. 레이아웃 설계
  4. 반응형 웹 디자인

JavaScript 프로그래밍

JavaScript는 웹 페이지의 동적 기능을 구현하는 프로그래밍 언어입니다.

// 간단한 JavaScript 예제
function sayHello() {
  alert("안녕하세요!");
}
